快速上手（中文）

  • 首页点 “+” 添加项目；填写 名称 与 到期日，可选：生产日期、备注、分类。
  • 设置 → 分类：新增/删除/重排分类并选择图标；删除分类会把相关项目设为“无分类”。
  • 导出 .zip（含 manifest.json + 图片）；导入会按“分类名称”匹配/创建。
  • 设置 → 提醒时间：选择每天推送的小时。若未收到通知，请在系统设置开启通知。
  • 清除：仅删除当前筛选结果里“显示”的项目，请谨慎操作。

FAQ (English)

  • Tap “+” to add; enter name and expiry date, then Save.
  • Settings → Categories (add/delete/reorder; icons supported).
  • Export .zip (manifest + images). Import creates missing categories by name.
  • Settings → Notification hour; enable app notifications in OS settings.
